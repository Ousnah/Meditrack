# Frontend

The frontend is built with Vue.js and Vite for a fast development experience and modular component structure. The UI is designed to be accessible and user-friendly, helping users track pre-surgery steps easily.

- **Framework:** Vue.js
- **Build Tool:** Vite
- **Focus:** Accessibility and clarity for all users.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
